# Gemini AI Provider

## 設定

```yaml
ai:
  provider: gemini
  model: gemini-2.0-flash
```

```bash
export AI_API_KEY=AIza...   # Google AI Studio 取得
```

## SDK

`github.com/google/generative-ai-go/genai`

## 可用 Model

| Model ID | 說明 |
| :--- | :--- |
| `gemini-2.0-flash` | 推薦，快速 |
| `gemini-2.0-pro` | 更強的推理能力 |
